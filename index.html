痛点

AI 编程助手的结构性缺陷

AI 对项目的了解局限于当前会话中的上下文。它不知道项目的技术规范、历史决策、踩过的坑——除非你每次都手动告诉它。

01

规范无法持久化注入

每次新对话都要重新粘贴项目规范（鉴权方案、命名约定）。AI 可能在不知道规范的情况下生成不符合项目约定的代码，增加 Code Review 负担。

02

架构决策无法被 AI 利用

历史决策（为什么选 MyBatis 而不是 JPA）散落在聊天记录/Wiki。AI 可能反复提出已被否决的方案，缺乏历史决策的上下文意识。

03

AI 代码逐渐偏离项目规范

多次对话中 AI 生成的代码缺少持久化的规范约束，风格逐渐漂移。当前唯一的对抗手段是 Code Review，人力成本持续攀升。

04

现有方案依赖人工维护

CLAUDE.md 思路正确但需人工编写，随项目规模增长维护成本不可持续。人工维护的内容容易与代码实际状态脱节，缺乏自动验证机制。

知识保真体系

校准 + 检测 + 仲裁 + 防腐

知识的价值取决于它的准确性。KnowForge 从四个维度构建知识保真体系——校准引擎确保代码-知识一致、五层检测覆盖所有冲突类型、证据仲裁防止低质量知识压制高价值知识、防腐烂机制防止知识静默过期。

🎯 智能校准引擎

8 种触发事件驱动的校准引擎——不只是被动记录，而是主动检测知识是否仍然正确。

8 种触发事件：Git commit、新服务上线、人肉修改（author ≠ AI）、需求文档变更、架构评审通过、API 变更、依赖升级、故障复盘

LLM 语义对比流程：
代码变更 → 查找关联知识（类级别映射）→ LLM 判断一致性
  • CONSISTENT → code_verified=1，校准时间重置
  • INCONSISTENT → 生成 UPDATE_CANDIDATE，旧知识标记 suspected_stale
  • UNCERTAIN → 三级兜底：注入警告 → 累积升级 → 长期未验证降级

关键约束：类级别校准是辅助信号，不直接改变知识状态。code_verified=1 时冷知识不受时间衰减惩罚——代码未变，知识就不需要重新验证。

🔍 五层冲突检测 L0-L5

从内容哈希到人工校准，层层递进——不只是检测新知识 vs 已有知识，还包括已有知识间的交叉扫描和知识-代码一致性检查。

L0 内容哈希精确匹配（MD5 归一化） 写入时
L1 语义签名哈希 + 假冲突过滤（单位归一化） 写入时
L2 LLM 矛盾判断（五分类：互斥/精炼/条件兼容/互补/等同） 写入时 + dream
L3 交叉扫描——同 domain 内已有知识间矛盾检测 dream 执行
L4 知识-代码语义一致性（LLM 对比知识描述 vs 代码行为） 代码变更
L5 人工校准——最终仲裁 按需

⚖️ 证据仲裁机制

矛盾了怎么判？不只比 confidence——比证据的可信度层级。代码不会撒谎，但 dead code 不会被当作权威。

Level 5 活代码 1.0 通过活性检查的可执行代码事实
Level 4 配置/文档 0.9 可验证的配置文件、API 文档
Level 3 用户陈述 0.7 用户明确表达的事实
Level 2 隐式推断 0.5 从对话上下文推断
Level 1 LLM 推理 0.3 LLM 从代码/文档推断
Level 0 无证据 0.0 LLM 凭空生成
仲裁规则：score = evidence_weight × confidence。差值 ≥ 0.30 自动采用（被替代知识进入 quarantined/ 隔离区观察 30 天），差值 < 0.10 必须人工裁决。自动采用的知识注入时附带可信度警告标记。

六类冲突各走不同策略：事实冲突证据仲裁、时效冲突版本标注、范围冲突例外补充、粒度冲突精炼取代、隐式冲突人工确认、人为冲突代码优先。

🔄 防腐烂体系

知识库最大的敌人不是缺失，而是过期。三层腐烂检测 + 晋升-修剪解耦 + 物理目录生命周期——确保知识持续保鲜。

静默过期
代码已变知识未更新
→ L4 签名校验 + Git diff
层级漂移
架构决策降级为历史背景
→ S2 长期未校验自动检测
覆盖盲区
新模块上线知识库为零
→ 文件覆盖率每周审计
晋升-修剪解耦：used_count 不参与晋升评分（冷知识不应因此被压制），但参与修剪规则（低频+低质+代码变更→候选删除）。code_verified=1 的知识受保护——冷但正确，留着。

晋升公式 V2.1：confidence × 0.70 + anchor_bonus × 0.15 + calibration_recency × 0.15。正确性权重占 7 成，校准时效仅辅助信号。

三目录生命周期：
staging/ 待确认 → knowledge/ 已采纳 → deprecated/ 已废弃
绿色通道：confidence ≥ 0.95 → 直接晋升 knowledge/。CANDIDATE 滞回：0.82 进 / 0.80 出。quarantined/ 隔离区：自动裁决被替代的知识保留 30 天可回滚。
